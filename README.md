# sterngold-discussions

Discussions repository powering comments on [sterngold.nl](https://sterngold.nl) via [Giscus](https://giscus.app).

This repo must remain public for Giscus to function.

## Security And Moderation

Do not post private data, tokens, credentials, or vulnerability reports in public
discussions. Use the reporting path in [SECURITY.md](SECURITY.md) for security,
spam, or moderation-bypass concerns.
