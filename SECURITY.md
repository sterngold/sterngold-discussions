# Security Policy

## Reporting a Vulnerability

This repository is public because Giscus needs it for site comments. Please do
not open public issues for security problems, spam campaigns, moderation bypasses,
or accidental private-data disclosure. Use GitHub private vulnerability reporting
if it is available, or email the maintainer listed on the profile.

Include:

- A short description of the issue.
- A link to the affected discussion or comment, if applicable.
- Reproduction steps or evidence.
- Whether private data or account compromise may be involved.
